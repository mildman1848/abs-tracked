# Contributing

English: [CONTRIBUTING.md](CONTRIBUTING.md)

## Vor einem Pull Request

- Änderungen sollten sich auf das Container-Image, die gebuendelten Services oder den Release-Workflow konzentrieren.
- Aktualisiere `README.md`, `README.DE.md` und die Changelog-Dateien, wenn sich Verhalten ändert.
- Prüfe Build-, Laufzeit- und Sicherheitsänderungen vor dem Push.
